# Core Engine Algorithm Overview

## 목적
사주 계산 핵심 엔진의 처리 단계와 품질 기준을 정의한다.

## 처리 흐름
1. 입력 정규화: 생년월일/시간/성별/타임존 정합
2. 시간 보정: 태양시 및 보정 규칙 적용
3. 운세기초 산출: 십간십이지, 음양, 오행
4. 분석 단계: 오행/십성/신살/형국/운세 주기
5. 결과 조합: 화면/콘텐츠에서 사용할 표준 도메인 객체로 변환

## 품질 기준
- 계산 경계값: 경계일/윤달/시간대 이슈 테스트
- 빈값/잘못된 입력 시 실패 처리
- 서버측 재현 로그 최소 항목(입력 hash, 결과 코드)

## 운영 연동
- API/페이지에서 동일 규격 사용 여부 점검
- 캐시 정책과 정합이 맞지 않을 경우 재계산 플래그 사용
