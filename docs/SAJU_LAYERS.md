# Secret Saju - Saju Layer System Architecture

**철학**: 정통 사주 명리학 기반 + Y2K 게임 감성

## 🏗️ Layer 0: Foundation (기초 빌딩 블록)

### 천간 (天干) - 10개
```
甲(갑) - 양목 - 큰 나무 - 리더십, 성장
乙(을) - 음목 - 풀, 덩굴 - 유연성, 적응
丙(병) - 양화 - 태양 - 열정, 카리스마
丁(정) - 음화 - 촛불 - 섬세함, 예술
戊(무) - 양토 - 산 - 안정, 신뢰
己(기) - 음토 - 흙 - 포용, 배려
庚(경) - 양금 - 쇠 - 강인함, 원칙
辛(신) - 음금 - 보석 - 예민함, 완벽주의
壬(임) - 양수 - 바다 - 지혜, 유동성
癸(계) - 음수 - 이슬 - 감수성, 직관
```

### 지지 (地支) - 12개
```
子(자) - 쥐 - 겨울 시작 - 지혜, 계산
丑(축) - 소 - 겨울 중반 - 성실, 끈기
寅(인) - 호랑이 - 봄 시작 - 활동, 추진력
卯(묘) - 토끼 - 봄 중반 - 섬세함, 평화
辰(진) - 용 - 봄 끝 - 변화, 권력
巳(사) - 뱀 - 여름 시작 - 지혜, 신비
午(오) - 말 - 여름 중반 - 열정, 활발
未(미) - 양 - 여름 끝 - 온순, 예술
申(신) - 원숭이 - 가을 시작 - 영리, 적응
酉(유) - 닭 - 가을 중반 - 계획, 정확
戌(술) - 개 - 가을 끝 - 충성, 보호
亥(해) - 돼지 - 겨울 시작 전 - 포용, 지혜
```

**현재 구현 상태**: 60갑자 (천간 × 지지) ✅

## 🎨 Layer 1: 오행 (五行) - 관계의 기초

### 오행 속성
```javascript
const WUXING = {
  木: { name: "나무", color: "green", trait: "성장" },
  火: { name: "불", color: "red", trait: "확장" },
  土: { name: "흙", color: "yellow", trait: "안정" },
  金: { name: "금속", color: "white", trait: "수렴" },
  水: { name: "물", color: "blue", trait: "지혜" }
};
```

### 오행 관계
```
상생(相生) - 도와주는 관계
木 → 火 → 土 → 金 → 水 → 木

상극(相剋) - 제약하는 관계
木 → 土 → 水 → 火 → 金 → 木
```

**게임 비유**: 포켓몬 타입 상성
- 나무가 불을 만나면: "열정 폭발" (상생)
- 나무가 금속을 만나면: "도끼에 베임" (상극)

**활용**:
- 궁합 계산 기초
- 직장/연애 관계 분석
- "이 사람은 나한테 상생인가 상극인가?"

## ⭐ Layer 2: 십성 (十星) - 관계의 핵심

### 십성 정의
```
비견(比肩) - 나와 같은 존재 - 친구이자 경쟁자
겁재(劫財) - 나와 비슷하지만 다름 - 라이벌
식신(食神) - 내가 만든 것 - 여유, 창작
상관(傷官) - 내가 표현하는 것 - 비판, 재능
편재(偏財) - 흘러가는 돈 - 부동산, 투자
정재(正財) - 고정된 돈 - 월급, 안정
편관(偏官) - 나를 억압하는 힘 - 상사, 압박
정관(正官) - 나를 이끄는 힘 - 명예, 직위
편인(偏印) - 불규칙한 배움 - 특수 기술
정인(正印) - 정통 학문 - 학위, 자격증
```

### 십성으로 관계 정의
```javascript
// 예시: 갑자(甲子) vs 병인(丙寅)
{
  "나": "甲_木",
  "상대": "丙_火",
  "십성": "식신",
  "관계": {
    "직장": "내가 만든 성과가 빛을 발함",
    "연애": "내가 표현하면 상대가 받아줌",
    "점수": 85
  }
}
```

**활용**:
- 60 × 60 = 3,600가지 관계 매트릭스
- 직장 상사와의 관계, 연인과의 관계 구체화
- "우리 상사는 편관이라 압박이 심해"

## 🌟 Layer 3: 신살 (神煞) - 특수 능력

### 주요 신살 목록

#### 1. 도화살 (桃花煞)
```
발동 조건: 지지에 子午卯酉 포함
효과: 이성운 MAX, 매력 증가
부작용: 바람기, 관계 복잡
```

#### 2. 역마살 (驛馬煞)
```
발동 조건: 지지에 寅申巳亥 포함
효과: 이동, 해외, 변화
부작용: 정착 어려움, 불안정
```

#### 3. 화개살 (華蓋煞)
```
발동 조건: 지지에 辰戌丑未 포함
효과: 예술, 종교, 학문
부작용: 외로움, 고독
```

#### 4. 천을귀인 (天乙貴人)
```
발동 조건: 천간-지지 특정 조합
효과: 귀인 만남, 위기 탈출
```

#### 5. 공망 (空亡)
```
발동 조건: 특정 갑자 조합
효과: 무(無)의 에너지
부작용: 상실, 비움
```

**게임 비유**: 패시브 스킬
- "이 캐릭터는 도화살 보유, 크리티컬 확률 +30%"
- "역마살 보유, 이동속도 +50%, 하지만 방어력 -20%"

## 🔬 신뢰성 검증 기준

1. **사주 명리학 원칙 준수**
   - 모든 계산은 전통 명리학 공식 사용
   - 천간-지지 조합 정확성
   - 오행 상생/상극 관계 준수

2. **데이터 출처**
   - 명리학 서적 기반
   - 검증된 만세력 데이터
   - 실제 사주 해석 사례 참고

3. **확장 시 경계**
   - ✅ OK: 오행/십성/신살 기반 해석
   - ✅ OK: 전통 개념의 현대적 표현
   - ❌ NO: 근거 없는 창작 개념
   - ❌ NO: 타로/점성술 등 타 체계 혼합
