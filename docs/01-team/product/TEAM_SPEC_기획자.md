# 🎯 기획자/PM 스펙 문서

## 프로젝트 개요

**프로젝트명**: Secret Paws (멍냥의 이중생활)  
**목표**: 사주아이(20만 유저)를 넘어서는 복사 불가능한 바이럴 사주 서비스  
**차별점**: 개인 사주 → **관계 지도** (엄마, 애인, 상사 등 다중 분석)

---

## 1. 비즈니스 모델

### 수익화 전략: Jelly 가상 화폐 시스템

| 상품 | 젤리 수량 | 보너스 | 가격 | 단가 | 타겟 |
|------|----------|--------|------|------|------|
| 맛보기 | 1개 | - | 990원 | 990원 | 첫 구매 유도 |
| 똑똑이 | 3개 | +1 (총 4개) | 2,900원 | 725원 | 나+애인 |
| **프로** | 10개 | +3 **(총 13개)** | **9,900원** | **762원** | **가족 전체** ⭐ |

**핵심 심리**:
- 990원 = 부담 없음 (사주아이와 동일)
- 9,900원 = "어차피 가족들 다 볼 건데 한번에 사는 게 싸네" (AOV↑)

---

### 전환 퍼널

```
① 회원가입 (무료)
   ↓
② 첫 사주 입력 (나) → 무료 ✅
   "당신은 대장 안 시켜주면 삐지는 독재자"
   ↓
③ 호기심 유발
   "엄마와 당신이 싸우는 진짜 이유가 궁금하신가요?"
   ↓
④ 2번째 프로필 추가 시도 (엄마)
   → 젤리 부족 → 샵 모달 노출 💳
   ↓
⑤ 젤리 구매 (평균 AOV: 5,500원)
   - 신규: 똑똑이 팩 (2,900원)
   - 파워유저: 프로 팩 (9,900원)
   ↓
⑥ 관계 대시보드 언락
   "엄마 23%, 애인 78%, 상사 45%"
   ↓
⑦ VS 모드 공유 → 바이럴 🔥
   "나 vs 전남친 복연 가능성 12%"
```

## 2. 핵심 기능 (Feature Spec)

### 2.1 관계 대시보드 (Killer Feature)

**사주아이와의 차별점**:
- 사주아이: 개인 사주만 (1회성)
- Secret Paws: **모든 관계를 한눈에** (중독성)

**구성**:
```
┌────────────────────────────┐
│      [나] 갑자일주          │
│         /  |  \            │
│       /    |    \          │
│    엄마   애인   상사       │
│    23%   78%   45%        │
│    🔥    💕    ⚠️         │
│                            │
│ 💡 인사이트:               │
│ • 엄마와 거리 두면 연애운↑  │
│ • 상사는 상극이지만 승진 OK │
└────────────────────────────┘
```

**Jelly 소비**:
- 관계 추가: 1 Jelly
- 상세 분석 열람: 1 Jelly
- VS 모드 비교: 2 Jellies

---

### 2.2 VS 모드 (Viral Engine)

**타겟 시나리오**:
- "나 vs 전남친" → 복연 가능성 분석
- "나 vs 현남친" → 장기 관계 전망
- "나 vs 상사" → 권력 다이나믹 분석

**공유 메커니즘**:
1. VS 결과를 인스타 스토리 형식으로 자동 생성
2. 워터마크 포함 ("Secret Paws에서 확인하기 >")
3. 공유 시 **1 Jelly 보상** (Share-to-Earn)
4. 친구 가입 시 **양측 +2 Jelly** (Referral)

---

### 2.3 오늘의 팩폭 (DAU Driver)

**사주아이 한계**: 1회 보고 끝  
**Secret Paws 해결책**: 매일 새로운 인사이트

**예시**:
```
[오늘의 팩폭]
갑자일주의 2026.01.31

"오늘 누가 뭐라고 해도 참아.
너 오늘 멘탈 종이야."

💊 처방: 퇴근 후 혼자 맥주 한 캔만.
⏰ 23시간 47분 후 새로운 팩폭 공개
```

**효과**: 
- 매일 아침 앱 확인 습관 → DAU 30%+
- 친구들한테 공유 → 자연 바이럴
