# CFO Financial Overview

## 목적
이 문서는 현금흐름, 결제 전환, 보유 잔액 소모율을 기준으로 의사결정을 지원한다.

## Unit Economics
- 가격 전략: `990 / 2,900 / 9,900` (원 단위)
- 핵심 지표:
  - 결제 전환율
  - 잔액 소비율
  - 첫 구매 → 재구매 전환율
  - LTV/CAC

## 분기 운영 체크
- MAU/매출/리텐션 지표를 분기 단위로 갱신한다.
- 결제 플로우 이상 징후(실패율 급증, 환불율 상승)를 월 1회 이상 점검한다.
- `active-dispatch.md`와 연계해 투자 우선순위(마케팅/인프라/콘텐츠)를 조정한다.

## 리스크 관리
- 결제 실패율 상승: 결제 UX 개선 또는 수수료/키 정책 점검
- 고정비 증가: 광고비와 API 비용 비율의 상한값 적용
- 이탈률 급증: 리텐션 기능(재방문/루틴) 강화와 연계
