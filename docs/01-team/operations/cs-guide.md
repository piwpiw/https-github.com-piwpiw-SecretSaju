## 🎯 CS 원칙

### 1. **빠른 응답** (Response Time)
- **목표**: 평균 2시간 이내 첫 응답
- **긴급**: 결제/계정 문제 → 1시간 이내
- **일반**: 사용법 질문 → 4시간 이내

### 2. **친절하고 명확한 톤**
```
❌ 나쁜 예:
"그건 안 됩니다."

✅ 좋은 예:
"안녕하세요! 현재는 해당 기능이 지원되지 않지만, 
고객님의 의견을 제품팀에 전달드리겠습니다. 
불편을 드려 죄송합니다."
```

### 3. **문제 해결 우선**
- 단순 답변보다는 **해결책 제시**
- 불가능한 경우 → 대안 제시
- 버그 발견 시 → 즉시 개발팀에 에스컬레이션

## 🏷️ 문의 유형별 대응

### 1. 결제 문제 ⚠️ (긴급)

#### 1-1. "결제했는데 젤리가 안 들어왔어요"

**확인 사항**:
1. Supabase `jelly_transactions` 테이블 확인
   ```sql
   SELECT * FROM jelly_transactions 
   WHERE user_id = '<user_id>' 
   ORDER BY created_at DESC;
   ```
2. Toss Payments 관리자 페이지에서 거래 확인

**Case A**: 거래 성공 but 젤리 미지급
→ **수동 지급** (개발자에게 요청)

**Case B**: 거래 실패
→ 재결제 안내 + 실패 원인 설명

**응답 템플릿**:
```
안녕하세요, Secret Saju입니다.

결제 내역을 확인해보니 [상황 설명].
불편을 드려 정말 죄송합니다.

[해결 방법]:
- 즉시 젤리를 수동 지급해드렸습니다. (Case A)
- 또는: 재결제를 진행해주시면 감사하겠습니다. (Case B)

추가 문의사항이 있으시면 언제든지 연락주세요.
감사합니다.
```

#### 1-2. "환불 요청"

**정책**:
- 구매 후 7일 이내: 100% 환불 (미사용 시)
- 젤리 사용 후: 부분 환불 (남은 젤리만)

**확인 사항**:
1. 구매 일자 확인
2. 젤리 사용 내역 확인

**프로세스**:
1. 환불 승인 → CTO/CEO에게 알림
2. Toss Payments 관리자에서 환불 처리
3. 사용자에게 확인 이메일 발송

### 2. 계정 문제

#### 2-1. "로그인이 안 돼요"

**체크리스트**:
- [ ] 카카오 로그인 버튼 클릭했는지
- [ ] 카카오 계정 연동 허용했는지
- [ ] 브라우저 쿠키 차단 설정 확인

**해결 방법**:
1. 브라우저 캐시 삭제
2. 시크릿 모드에서 시도
3. 다른 브라우저로 시도
4. 여전히 안 되면 → 개발팀 에스컬레이션

#### 2-2. "계정 삭제 요청"

**확인**:
- 정말 삭제하시겠습니까?
- 저장된 프로필, 젤리 모두 삭제됩니다.

**프로세스**:
1. 사용자 확인
2. 개발팀에 요청 → Supabase에서 삭제
3. 삭제 완료 이메일 발송

**법적 요구사항**: GDPR/개인정보보호법 준수

### 3. 사용법 문제

#### 3-1. "사주 계산 결과가 이상해요"

**확인 사항**:
- 생년월일 정확히 입력했는지
- 음력/양력 선택 확인
- 출생 시간 정확한지 (모를 경우 "모름" 선택)

**응답**:
```
안녕하세요!

사주 계산은 다음 정보를 기반으로 합니다:
- 생년월일: [확인]
- 음력/양력: [확인]
- 출생 시간: [확인]

혹시 위 정보가 정확하지 않으시다면 다시 입력해주세요.
만약 정확한데도 결과가 이상하다면, 구체적으로 어떤 부분이 
이상한지 알려주시면 확인해드리겠습니다.

감사합니다!
```

#### 3-2. "프로필을 삭제하고 싶어요"

**단계**:
1. 마이페이지 이동
2. 프로필 목록에서 해당 프로필 선택
3. "삭제" 버튼 클릭

**주의**: 삭제 시 복구 불가능

#### 3-3. "젤리는 어떻게 사용하나요?"

**응답**:
```
젤리는 프리미엄 기능을 해금하는 데 사용됩니다:

- 상세 분석 (10년 대운, 신살 등): 1 젤리
- 궁합 분석: 1 젤리  
- 셀럽 매칭 (향후): 1 젤리

젤리는 다음과 같이 구매하실 수 있습니다:
- TRIAL: 990원 = 1 젤리
- SMART: 2,900원 = 4 젤리 (1개 보너스)
- PRO: 9,900원 = 13 젤리 (3개 보너스)

감사합니다!
```

### 5. 버그 신고

**중요도 분류**:

| 중요도 | 예시 | 대응 시간 |
|--------|------|-----------|
| **Critical** | 결제 실패, 로그인 불가 | 즉시 (1시간) |
| **High** | 계산 오류, 데이터 손실 | 당일 |
| **Medium** | UI 깨짐, 느린 로딩 | 3일 |
| **Low** | 텍스트 오타, 디자인 이슈 | 1주 |

**프로세스**:
1. 재현 단계 확인
2. 스크린샷/녹화 요청
3. GitHub Issue 생성 (라벨: `bug`)
4. 개발팀 할당
5. 수정 완료 → 사용자에게 감사 메시지

**응답**:
```
버그 신고 감사합니다!

확인 결과 실제 버그로 확인되었습니다.
개발팀에 전달하여 최대한 빨리 수정하겠습니다.

불편을 드려 죄송하며, 
수정 완료 시 다시 알려드리겠습니다.

감사합니다!
```

## 🔍 FAQ (자주 묻는 질문)

### Q1. 사주가 정확한가요?
**A**: Secret Saju는 진태양시, 음력 변환 등을 적용한 
고정밀 계산을 사용합니다. 오프라인 전문가와 동일한 
정확도를 목표로 합니다.

### Q2. 무료로 어디까지 볼 수 있나요?
**A**: 기본 사주 (사주팔자 4기둥, 오행 분석)는 무료입니다.
상세 분석, 궁합 등은 젤리가 필요합니다.

### Q3. 출생 시간을 모르면 어떻게 하나요?
**A**: "시간 모름"을 선택하시면 시주(時柱)를 제외한
연주, 월주, 일주로 분석해드립니다.

### Q4. 환불이 가능한가요?
**A**: 구매 후 7일 이내, 젤리 미사용 시 100% 환불 가능합니다.

### Q5. 개인정보는 안전한가요?
**A**: 모든 데이터는 암호화되어 저장되며,
제3자와 공유하지 않습니다. (자세한 내용은 개인정보처리방침 참고)
