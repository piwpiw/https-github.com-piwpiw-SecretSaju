## 📚 전통 명리학의 수치화 방법론

### 1. 일간 강약 점수 (日干强弱)

**공식**: 득령 + 득지 + 득세 = 100점 만점

#### A. 득령 (得令, 30점)
- **정의**: 일간이 태어난 계절(월지)에서 힘을 얻는가?
- **배점**:
  - 왕지(旺地): 30점 - 일간과 같은 오행의 계절
  - 상지(相地): 20점 - 일간을 생하는 오행의 계절
  - 휴지(休地): 10점 - 일간이 생하는 오행의 계절
  - 수지(囚地): 5점 - 일간을 극하는 오행의 계절
  - 사지(死地): 0점 - 일간이 극하는 오행의 계절

**예시**:
```
甲木(봄 태생) → 30점 (왕지)
甲木(여름 태생) → 10점 (휴지, 木生火)
甲木(가을 태생) → 0점 (사지, 金克木)
甲木(겨울 태생) → 20점 (상지, 水生木)
```

#### B. 득지 (得地, 30점)
- **정의**: 지지(4개)에 뿌리가 몇 개 있는가?
- **배점**: 뿌리 1개당 7.5점 (최대 4개 = 30점)
- **뿌리 기준**:
  - 동일 오행
  - 생해주는 오행
  - 지장간(支藏干) 포함

**예시**:
```
甲木 일간 + 지지에 寅(木), 亥(水) → 2개 뿌리 = 15점
```

#### C. 득세 (得勢, 40점)
- **정의**: 천간(4개) 중 도와주는 글자가 몇 개인가?
- **배점**: 도움 1개당 10점 (최대 4개 = 40점)
- **도움 기준**:
  - 비겁(同): 같은 오행
  - 인성(生): 나를 생하는 오행

**예시**:
```
甲木 일간 + 천간에 乙(木), 壬(水) → 2개 도움 = 20점
```

### 2. 신강신약 판정 (身强身弱)

**기준**:
- **60점 이상** = 신강(身强) - 일간이 강함
- **40~59점** = 중화(中和) - 균형
- **40점 미만** = 신약(身弱) - 일간이 약함

**의미**:
- 신강 → 설기(泄氣)가 필요 (식상, 재성 좋음)
- 신약 → 보강(補强)이 필요 (인성, 비겁 좋음)

---

### 3. 격국 판정 (格局判定)

**정격(正格)** - 일반적인 사주:
1. **정관격** - 월지에 정관
2. **정재격** - 월지에 정재
3. **식신격** - 월지에 식신
4. **인수격** - 월지에 정인

**외격(外格)** - 특수한 사주:
1. **종격** - 일간이 극도로 약함 → 따라감
2. **화격** - 특정 오행으로 변화
3. **전왕격** - 일간이 극도로 강함

---

### 4. 용신 선정 (用神選定)

**용신**: 사주에서 가장 필요한 오행

**선정 기준**:
- 신강 → 극설(克泄) 오행이 용신 (재성, 식상, 관성)
- 신약 → 생조(生助) 오행이 용신 (인성, 비겁)

**점수화**:
- 용신 오행이 강하면 → 운이 좋음 (80점+)
- 용신 오행이 약하면 → 운이 나쁨 (30점-)

## 📊 최종 점수 체계

### 종합 운세 점수
```
최종 점수 = (일간 강약 × 0.3) + (격국 완성도 × 0.4) + (용신 강도 × 0.3)
```

**등급**:
- 90-100점: S급 - 타고난 운
- 80-89점: A급 - 좋은 운
- 70-79점: B급 - 평균 이상
- 60-69점: C급 - 평균
- 50-59점: D급 - 평균 이하
- 50점 미만: F급 - 노력 필요
