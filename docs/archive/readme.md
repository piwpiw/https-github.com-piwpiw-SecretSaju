## Archive 정리

- 중복 문서 경로를 식별해 `docs/ARCHIVE` 또는 해당 상태 레이블로 이전

## Archive Dedup Rule
- If a doc exists in `docs/archive` and active tree, keep only one active source and mark the other as archived.
- Add a short pointer line in the archived copy linking to the active doc.
- Avoid duplicate templates; consolidate to a single canonical template.
