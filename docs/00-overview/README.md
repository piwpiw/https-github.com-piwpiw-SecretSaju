# SecretSaju Overview

## Project Summary
SecretSaju is a relationship-centric fortune platform built on Next.js + TypeScript.
The current phase focuses on fast execution, safe deployment, and synchronized documentation for multi-agent team operation.

## Current Operational Focus (2026-03-01)
- Route/page contracts are centralized in `execution-backlog-ko.md`.
- Deployment prechecks are optimized with parallel local preflight.
- Team instruction speed is improved by SOT-first documentation flow.

## Core Product Pillars
1. Accurate fortune analysis pipeline.
2. Relationship and comparison experiences for retention.
3. Paid conversion flow with wallet/payment reliability.
4. Operational visibility through admin/support routes.
