# Advanced Compatibility System (Gung-hap) - Implementation Plan

**목표**: 단순한 오행 상생상극(木생火 등)을 넘어, **지지 합충형파해(合沖刑破害)**와 **천간 합충(合沖)**을 적용한 정밀 궁합 시스템 구현.

---

## 🎯 문제 정의

### 현재 상태 (10인 회사)
- ❌ 단순 오행 궁합 (예: 물과 불은 상극)
- ❌ 단편적인 판단 (좋다/나쁘다 이분법)
- ❌ 디테일 부족 (왜 좋은지 설명 불가)

### 목표 상태 (100인 회사)
- ✅ **천간합(天干合)**: 정신적 교감, 이끌림 (예: 甲己合)
- ✅ **지지삼합(地支三合)/육합(六合)**: 현실적 조화, 속궁합, 사회적 결합
- ✅ **상충(相沖)**: 강렬한 끌림과 충돌 (예: 子午沖)
- ✅ **원진살(怨嗔殺)**: 애증 관계, 집착
- ✅ **점수화**: 100점 만점의 정밀 스코어링

---

## 📚 전통 명리학 이론 적용

### 1. 천간 관계 (정신/이상 매칭)

#### A. 천간합 (무게 중심 15점)
- 甲 + 己 = 토(土) : 중정지합 (바른 사랑)
- 乙 + 庚 = 금(金) : 인의지합 (강단 있는 사랑)
- 丙 + 辛 = 수(水) : 위제지합 (권위와 화려함)
- 丁 + 壬 = 목(木) : 인수지합 (음란지합, 감성적)
- 戊 + 癸 = 화(火) : 무정지합 (냉정한 사랑)

#### B. 천간충 (무게 중심 -10점)
- 甲-庚, 乙-辛, 丙-壬, 丁-癸

---

### 2. 지지 관계 (속궁합/현실 매칭)

#### A. 육합 (개인적 조화, 무게 중심 20점)
- 子丑合, 寅亥合, 卯戌合, 辰酉合, 巳申合, 午未合

#### B. 삼합 (사회적 조화, 무게 중심 15점)
- 申子辰 (水국), 亥卯未 (木국), 寅午戌 (火국), 巳酉丑 (金국)

#### C. 방합 (형제/가족애, 무게 중심 10점)
- 寅卯辰, 巳午未, 申酉戌, 亥子丑

#### D. 충/원진 (부정적 요소, 하지만 "강렬함"으로 해석)
- **충(沖)**: 子午, 丑未, 寅申, 卯酉, 辰戌, 巳亥 (-15점 or 자극적 관계)
- **원진(怨嗔)**: 子未, 丑午, 寅酉, 卯申, 辰亥, 巳戌 (-20점, 집착/애증)

## 🔬 점수 로직 (Scoring Logic)

**기본 점수**: 50점 (시작)

| 요소 | 점수 변동 | 설명 |
| :--- | :--- | :--- |
| **천간합** | +15 | 정신적인 코드가 잘 맞음 |
| **천간충** | -10 | 가치관 충돌 가능성 |
| **지지 육합** | +20 | 속궁합 최고, 찰떡궁합 |
| **지지 삼합** | +15 | 함께 무언가 도모하기 좋음 |
| **지지 충** | -10 ~ +5 | 싸우면서 정드는 관계 (케바케) |
| **지장간 통근** | +10 | 서로가 서로에게 뿌리가 됨 |
| **원진살** | -20 | 이유 없이 미움, 집착 |
| **조후(계절)** | +10 | 뜨거운 사주 + 차가운 사주 (보완) |

## 🏗️ 구현 설계

### Phase 1: 관계 계산 라이브러리 (`compatibility.ts`)

```typescript
interface RelationshipResult {
  score: number;       // 0-100
  level: string;       // S, A, B, C, F
  chemistry: {
    mental: number;    // 천간 기반 (정신)
    physical: number;  // 지지 기반 (육체/현실)
    conflict: number;  // 충/원진 기반 (갈등/자극)
  };
  tags: string[];      // ["#정신적지주", "#속궁합최고", "#애증의관계"]
  details: {
    type: "합" | "충" | "원진" | "평";
    name: string;      // "자오충", "갑기합"
    description: string;
  }[];
}
```

### Phase 2: 시뮬레이터 UI
- **Dual Selector**: 나(A) vs 상대방(B) 선택
- **Visual Matrix**: 관계도 시각화 (선 연결)
- **Score Dashboard**: 레이더 차트 (정신, 현실, 갈등, 지속성)

## 🔮 100인 회사급 디테일 전략

1. **"단점도 매력이다"**:
   - 단순히 충(沖)을 나쁘게만 보지 않고, "스파크가 튀는", "지루하지 않은" 관계로 해석.
2. **조후 해결**:
   - 여름생(火왕)과 겨울생(水왕)의 만남은 충이지만, 조후로는 "시원함"을 줌. 가산점 부여.
3. **일주(Day Pillar) 기준**:
   - 가장 강력한 일주론 기반 매칭 우선.
