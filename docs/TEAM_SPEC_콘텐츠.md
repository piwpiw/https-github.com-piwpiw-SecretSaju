# 📝 콘텐츠 팀 스펙

## 역할

사주아이를 넘어서는 **"복사 불가능한 콘텐츠 경쟁력"** 구축

---

## 1. 콘텐츠 구조

### 1.1 60개 캐릭터 (일주)

**현재 상태**: 3개 완료 (갑자, 을축, 병인)  
**목표**: 60개 전부 작성

**각 캐릭터당 필드**:

#### ① Persona (성격)
- `title`: "대장 안 시켜주면 삐지는 독재자"
- `subtitle`: "차가운 이성 뒤에 숨겨진 뜨거운 인정욕구"
- `summary`: 150자 내외 핵심 설명
- `tags`: #자존심킹, #인정욕구_빌런 등
- **`validation_hook`** ⭐ (신규 추가):
  ```json
  {
    "desc": "너 남한테 무시당한 거 같으면 며칠씩 곱씹지 않아? 혼자 있을 때 열불 터지는 거 딱 걸렸네.",
    "purpose": "사용자가 '어? 맞네?' 하게 만드는 관찰"
  }
  ```

#### ② Wealth Analysis (재물운)
- `type`: "귀 얇은 팔랑귀 투자자"
- `desc`: 돈 버는 방식/버릇 분석
- **`action_item`** ⭐ (신규 추가):
  ```json
  {
    "mission": "주식 앱 3일간 삭제하고 자격증 정보 검색하기",
    "reason": "너의 뇌는 '단기 도박'이 아니라 '장기 승부'에 최적화됨"
  }
  ```

#### ③ Love Analysis (연애운)
- `style`: "내로남불 선생님"
- `desc`: 연애 패턴 분석
- `ideal`: "당신의 징징거림을 받아주는 흙(土) 기운의 사람"

#### ④ Hidden Truth (숨겨진 진실)
- `blur_keyword`: "모친의 간섭" (카드에 blur 처리)
- `text`: 민감하지만 핵심적인 진실

---

### 1.2 관계 분석 콘텐츠

**관계 타입별 조언**:
- 엄마: "독립이 답", "경계선 설정"
- 애인: "밀당 금지", "기대치 낮추기"
- 상사: "말 줄이고 실적으로", "이직 타이밍"
- 전애인: "인스타 차단", "3개월 no contact"

**액션 아이템 예시**:
```
엄마 관계 개선
1. 이번 주말 전화 받지 말기
2. 독립 자금 월 30만원 모으기
3. 엄마 조언에 "들었어요"만 답하기
```

### 1.3 Daily Secret (오늘의 팩폭)

**목표**: 매일 새로운 인사이트로 **DAU 확보**

**포맷**:
```
[2026.01.31 갑자일주]

💬 "오늘 누가 뭐라 해도 참아.
    너 오늘 멘탈 종이야."

💊 처방: 퇴근 후 혼자 맥주 한 캔만 마시고 자.

⏰ 23시간 47분 후 새로운 팩폭 공개
```

**콘텐츠 규칙**:
- 최대 50자 (모바일 한 화면에)
- 반말 톤 (친근하지만 팩폭)
- 구체적 행동 지침 포함

**생성 방법**:
- Option A: 60개 × 365일 = 21,900개 미리 작성 (비현실적)
- **Option B**: 템플릿 + 변수 조합 (추천)
  ```
  Template: "오늘은 {상황}이 예상돼. {조언}."
  Variables:
    상황: ["감정 기복", "인간관계 스트레스", "재물 유혹"]
    조언: ["참고 넘어가", "거리 두기", "지갑 잠그기"]
  ```

## 2. 톤앤매너 (Tone & Manner)

### 2.1 핵심 원칙

**"귀엽지만 팩폭한다"**

- ❌ 점잖은 한자어: "귀하께서는 재물운이 미약하옵니다"
- ✅ 직설적 반말: "너 돈 모으는 재능 없어. 버는 족족 써"

**예시**:
```
나쁜 예: "당신은 감정 기복이 있으시니 자제하세요."
좋은 예: "너 작심삼일 DNA잖아. 인정해."
```

### 2.2 금기사항

**법적 리스크**:
- ❌ 의료 조언: "암 예후", "정신질환 진단"
- ❌ 투자 권유: "이 종목 사라", "부동산 지금 안 사면 후회"
- ❌ 자살/자해 암시: "죽고 싶을 거야" 등

**대신**:
- ✅ 성격 분석
- ✅ 관계 조언
- ✅ 라이프스타일 제안

## 4. 품질 관리

### 4.1 체크리스트

**각 캐릭터별**:
- [ ] Validation hook이 구체적인가? (추상적 X)
- [ ] Action item이 실행 가능한가? (모호함 X)
- [ ] 법적 리스크 없는가? (의료/투자 조언 X)
- [ ] 톤앤매너 일관적인가? (반말, 직설적)

**예시**:
```
❌ "너는 감정적이야" (추상적)
✅ "너 화나면 문 쾅 닫고 나가잖아" (구체적)
```

---

### 4.2 A/B 테스트 대상

**가설**:
- "엄마와의 궁합" vs "엄마와 싸우는 이유" → 어느 CTA가 더 끌리나?
- "990원 사주" vs "990원 운명 분석" → 어떤 워딩이 더 매력적?

**지표**:
- 클릭률
- 전환율
- 공유율
