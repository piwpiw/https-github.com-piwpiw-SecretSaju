# Growth Strategy

## Q2 Growth Levers
- Increase share-loop depth: result sharing, invite rewards, referral sequencing
- Strengthen daily routine: scheduled daily updates and reminder triggers
- Optimize conversion: A/B test CTA labels, CTA position, and message timing

## Execution Framework
- Funnel metric: Visit -> Analysis complete -> Payment -> Revisit
- Run experiments by channel, campaign, and page
- Weekly review with hypothesis validation and KPI rollback

## Core KPIs
- New-user conversion
- Weekly/monthly return rate
- Invite-based acquisition
