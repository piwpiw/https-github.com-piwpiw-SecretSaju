# Competitive Analysis

## 차별화 포인트
- 계산 정확성(천문/음력/태양시)과 해석의 일관성
- UI/동선 최적화를 통한 반복 사용 유도
- 공유형 성장 구조(Viral)와 결제 트리거 결합

## 위험군(취약점)
- 초기 퍼널의 온보딩 속도
- 결제/CS 응답속도
- 콘텐츠 업데이트 속도의 계절 편차

## 대응 전략
- 퍼널 단축 실험(A/B)과 가설 테스트
- 결제 오류 대응 문구 및 재시도 UX 개선
- 주기적 콘텐츠 캘린더 운영으로 시즌 변동 완화
